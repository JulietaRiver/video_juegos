Análisis de Ventas de Videojuegos

**Descripción del Proyecto**

Este proyecto tiene como objetivo analizar y predecir las ventas de videojuegos en diferentes plataformas y géneros. Se utilizaron técnicas de análisis de datos y aprendizaje automático para identificar patrones y tendencias en el mercado de los videojuegos.

**Alcance del Proyecto**

Carga y preprocesamiento de datos: Se recopilaron y limpiaron datos de ventas de videojuegos, eliminando valores nulos y duplicados.

Análisis exploratorio: Se examinaron tendencias de ventas por año, plataforma, género y región geográfica.

Visualización de datos: Se generaron gráficos para entender mejor la distribución de las ventas y los juegos más populares.

Modelado predictivo: Se entrenaron modelos de regresión para predecir las ventas de un videojuego en función de sus características.

Evaluación del modelo: Se midieron métricas de rendimiento como el MAE y R² para validar la precisión del modelo.

Resultados y Conclusiones

Se identificaron los géneros de videojuegos más vendidos y las plataformas con mayores ingresos.

Se observó una correlación entre la puntuación de los críticos y las ventas totales de un videojuego.

El modelo de regresión logró un R² aceptable, permitiendo predecir con cierta precisión las ventas futuras de videojuegos.

Se evidenció que los videojuegos con campañas de marketing más grandes tienden a generar mayores ventas en sus primeras semanas de lanzamiento.

**Tecnologías Utilizadas**

Python

Pandas y NumPy para manipulación de datos

Matplotlib y Seaborn para visualización

Scikit-learn para modelado predictivo
